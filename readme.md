# mini_chat_server

这是一个基于 Linux 网络编程实现的简易小型聊天服务器项目。

根据学习内容然后部分AI辅助下完成的一个小型聊天服务器， 主要目的是将知识点串联巩固学习内容， 并将知识转实践然后转换为真反馈。

这个项目目前实现了一个最基础的聊天室功能：服务端可以接收多个客户端连接，客户端发送的消息会被服务器接收并转发给其他在线客户端。客户端这边则可以一边监听用户输入，一边接收服务器发来的消息并显示在终端上。

## 实现的功能

目前已经完成的功能有：

- 基于 TCP 的客户端和服务端通信
- 服务端支持多个客户端同时连接
- 使用 `epoll` 进行 I/O 多路复用
- 客户端可以同时处理标准输入和网络消息
- 支持消息广播
- 支持客户端断开连接后的清理
- 客户端使用 `pipe + splice` 将输入内容转发到 socket

## 项目结构

- `server.cpp`  
  服务端代码，负责处理连接、接收消息和广播消息

- `client.cpp`  
  客户端代码，负责读取用户输入、发送消息以及接收服务器消息

- `makefile`  
  编译项目使用
